About This Project

Fitness Level Tracking is a locally-run, data-driven Windows Forms application designed for coaches and athletes. It enables tracking of multiple athletes through a tabbed interface, with comprehensive test metrics to monitor physical growth and performance over time. The application supports advanced filtering capabilities for in-depth self-study and performance analysis.

Key Features

  • Multi-athlete support with tabbed profile management
  • Comprehensive physical test metrics tracking
  • Growth visualization and progress monitoring
  • Advanced data filtering for self-study analysis
  • Local data storage for privacy and offline use
  • Clean, intuitive Windows desktop interface
